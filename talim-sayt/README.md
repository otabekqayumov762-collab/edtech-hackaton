# MF Platform

O'zbekiston abituriyentlari uchun **gamifikatsiyalangan, interaktiv va zamonaviy** imtihon tayyorgarlik platformasi. To'liq frontend (React + TypeScript + Vite + Tailwind v4).

## Imkoniyatlar

- **Marketing sayt** — slaydlardagi to'liq pitch (muammo, bozor, auditoriya, yechim, gamification, xalqaro tajriba, abt.uz taqqoslash, imkoniyatlar, jarayon, tariflar, roadmap, testimonial, CTA).
- **Ilova** — Dashboard, Darslar + dars sahifasi, Testlar + interaktiv test yechish + natija tahlili, Mashqlar (kunlik challenge), Reyting (podium + leaderboard), Statistika (grafiklar), Yutuqlar, AI yordamchi, Profil, Tariflar.
- **Gamifikatsiya** — XP, level (avtomatik level-up modal + konfetti), kunlik seriya (streak), kunlik maqsad, yutuqlar, reyting.
- **Demo foydalanuvchilar** — Saydalixon, Sevinch, Asadbek (login sahifasidan bir bosishda kirish). Holat `localStorage`da saqlanadi.

## Texnologiya

React 19 · TypeScript · Vite · Tailwind CSS v4 · React Router · Framer Motion · Recharts · Lucide icons

## Ishga tushirish

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build -> dist/
```
